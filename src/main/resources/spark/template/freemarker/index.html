Steganography: the practice of concealing messages or information within other nonsecret text or data.

This is an instructional page to showcase the steganography library I created for my computer networks class.

You should see two images below. The first one is the original and the second has your computers IP address encoded into it. If you can't see the difference, then it's working! Go ahead and save the image then run it through the decoder. It should spit out the IP from the computer which originally requested the picture.
You can download the decoder here. Place the image same directory as the decoder then enter the command "java -jar decoder.jar", enjoy!
